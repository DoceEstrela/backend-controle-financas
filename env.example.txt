# O sistema validará que tem pelo menos 32 caracteres em produção

# Configuração SMTP para envio de emails (Reset de Senha e Verificação)
# Se não configurado, o sistema funcionará em modo desenvolvimento (sem envio real)
#
# IMPORTANTE: Para evitar que emails caiam no spam:
# 1. Use um domínio próprio no SMTP_FROM (não use @gmail.com ou @hotmail.com)
# 2. Configure SPF, DKIM e DMARC no DNS do seu domínio
# 3. Verifique o domínio no seu provedor de email
# Veja EMAIL-DELIVERABILITY.md para instruções detalhadas
#
# Exemplo para Gmail (use Senha de App, não a senha normal):
# SMTP_HOST=smtp.gmail.com
# SMTP_PORT=587
# SMTP_SECURE=false
# SMTP_USER=seu-email@gmail.com
# SMTP_PASS=sua-senha-de-app-gmail
# SMTP_FROM=noreply@seudominio.com
# SMTP_FROM_NAME=Sistema de Vendas
# SMTP_REPLY_TO=suporte@seudominio.com
#
